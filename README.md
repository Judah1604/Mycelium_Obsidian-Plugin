# Mycelium

A second brain intended to grow into a large knowledge base that constantly refines itself, exposes weaknesses, discovers hidden connections, and strengthens ideas over time.

## Contributing

To make changes to this plugin, first ensure you have the dependencies installed.

```
pnpm install
```

### Development

To start building the plugin with what mode enabled run the following command:

```
pnpm run dev
```

_Note: If you haven't already installed the hot-reload-plugin you'll be prompted to. You need to enable that plugin in your obsidian vault before hot-reloading will start. You might need to refresh your plugin list for it to show up._

### Releasing

To start a release build run the following command:

```
pnpm run build
```
